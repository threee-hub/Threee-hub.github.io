采取hdfd命令来对FileSystem进行操作

  1. 创建文件夹
    hadoop fs -mkdir 文件名 分布式文件系统的目录 Eg：hadoop fs -mkdir /java/
  2. 上传文件
    hadoop fs -put Linnux文件下的文件 hdfs下面的文件
  3. 下载文件
    hadoop fs -get 分布式文件系统的文件 下载到的路径
  4. 删除文件或目录
    hadoop fs -rm 分布式文件系统的文件
    hadoop fs -rmr 分布式文件系统的目录
